Talk Selection Screen

The login screen is the first screen of the application

The screen allows you to:

  1. Select the talk you want to enter
  2. Specify credentials, if you are logging in for the first time.

In the Select talk combo box you can choose an existing, on-going talk, or enter a name to start a new one.
